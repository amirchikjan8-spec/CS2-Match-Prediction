# CS2-Match-Prediction
Проект по прогнозированию результатов матчей Counter-Strike 2 с использованием методов машинного обучения.

### Задача проекта

Целью проекта является разработка модели машинного обучения для прогнозирования результатов матчей в игре Counter-Strike 2 на основе исторических данных профессиональных турниров.

В рамках проекта выполняется сбор и анализ статистики матчей, предобработка данных, создание информативных признаков и обучение модели Random Forest для предсказания победителя матча. Модель использует данные о командах, рейтингах, винрейтах и других статистических показателях для определения наиболее вероятного исхода встречи.

Основная задача программной части проекта заключается в автоматизации процесса анализа данных и прогнозирования результатов матчей. Код проекта выполняет загрузку датасета, очистку данных, подготовку признаков, разделение выборки на обучающую и тестовую части, обучение модели, получение прогнозов и оценку качества работы модели с использованием метрик Accuracy, Precision, Recall и F1-score.

Результатом работы является обученная модель, способная прогнозировать победителя матча Counter-Strike 2 на основе имеющейся статистики и демонстрировать эффективность применения методов машинного обучения в области киберспорта.
