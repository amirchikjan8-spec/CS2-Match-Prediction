# CS2-Match-Prediction
Проект по прогнозированию результатов матчей Counter-Strike 2 с использованием методов машинного обучения.

### Задача проекта

Целью проекта является разработка модели машинного обучения для прогнозирования результатов матчей в игре Counter-Strike 2 на основе исторических данных профессиональных турниров.

В рамках проекта выполняется сбор и анализ статистики матчей, предобработка данных, создание информативных признаков и обучение модели Random Forest для предсказания победителя матча. Модель использует данные о командах, рейтингах, винрейтах и других статистических показателях для определения наиболее вероятного исхода встречи.

Основная задача программной части проекта заключается в автоматизации процесса анализа данных и прогнозирования результатов матчей. Код проекта выполняет загрузку датасета, очистку данных, подготовку признаков, разделение выборки на обучающую и тестовую части, обучение модели, получение прогнозов и оценку качества работы модели с использованием метрик Accuracy, Precision, Recall и F1-score.

Результатом работы является обученная модель, способная прогнозировать победителя матча Counter-Strike 2 на основе имеющейся статистики и демонстрировать эффективность применения методов машинного обучения в области киберспорта.

Датасет:
Изучив материал посвященный всем турнирам Tier 1,2,3 сцены я пришёл к следующим выводам.
Наиболее часто в датасете встречаются карты Mirage и Inferno. Некоторые карты представлены значительно реже, например Train, что может ограничивать качество прогнозов для данных карт.
Распределение побед между командами является относительно сбалансированным, что является благоприятным фактором для обучения модели машинного обучения и снижает риск появления смещения в сторону одного из классов.
Анализ статистики игроков показал, что более высокий показатель ADR (Average Damage per Round) связан с большей вероятностью победы команды. Это свидетельствует о том, что нанесённый урон является одним из важных факторов успеха в матче.
Игроки победивших команд в среднем демонстрируют более высокий показатель KAST, что указывает на более стабильный вклад в игровые раунды и эффективность командного взаимодействия.
Корреляционный анализ показал наличие положительной связи между количеством убийств (Kills), показателем ADR и разницей убийств и смертей (KDDiff). Это подтверждает, что данные характеристики отражают общую эффективность игрока.
Среди игроков наблюдается заметная разница в среднем количестве убийств за карту, что позволяет выделить наиболее результативных игроков и использовать их статистику как потенциально значимые признаки для прогнозирования результатов матчей.
Датасет содержит большое количество матчей, игроков, команд и турниров, что обеспечивает достаточный объём данных для дальнейшего построения и обучения моделей машинного обучения.
